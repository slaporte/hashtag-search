What is a hashtag?

We follow Twitter's specification (mostly) for hashtags: a hash mark (#) followed by one or more alphanumeric characters. Similarly, a mention is an at sign (@) followed by one or more alphanumeric characters. Hashtags shouldn't include any punctuation or only numbers.

Downloading results

You can download CSV results for a hashtag at http://tools.wmflabs.org/hashtags/csv/<tag>?limit=<limit>. If you do not provide a limit parameter, it will return 20,000 revisions by default. You can also optionally provide a lang parameter to limit your results to one version of Wikipedia. (You can also do lang=wikidata).

The columns in the CSV download are based on the RecentChanges table in the MediaWiki database. See the MediaWiki docs for more information on these fields.

Which languages do you support?

We currently support a few languages that are included in the search bar above. We are incrementally rolling out support for other languages. If you are running an edit-a-thon on a Wikipedia we don't currently support, open an issue on github. We may even be able to load past edits (within the previous 60 days).
